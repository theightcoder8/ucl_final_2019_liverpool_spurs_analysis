# UEFA Champions League Final 2019 – Match Analysis

This project analyzes the 2019 UEFA Champions League Final between Liverpool and Tottenham using event data and Python.

The goal is to explore attacking patterns, player involvement, and team structure through visual analytics.

## Project Components

The notebook includes several analyses:

• Shot Map – visualizing Liverpool's shooting locations and shot quality  
• Pass Network – average player positions and passing connections  
• Player Comparison – comparing Mohamed Salah and Heung-Min Son using attacking metrics  
• Expected Goals (xG) – measuring chance quality during the match

## Tools Used

Python  
Pandas  
Matplotlib  
NumPy  
StatsBomb Event Data

## Key Insight

Liverpool generated higher-quality scoring chances, particularly through central areas of the penalty box. Mohamed Salah produced the highest shot volume and expected goals in the match.
